# Setup Instructions

## General
1. You should have git installed
2. Clone the repository with "git clone https://github.com/derekchang4/stock-data-assessment.git"

## Backend
1. Install the Dotnet SDK from https://dotnet.microsoft.com/en-us/download
2. cd into the stock-data-assessmant/backend/api directory and run "dotnet watch run"
3. The API is now live on localhost:5037
* You can go to localhost:5037/swagger/index.htm to manually test the API

## Frontend
1. Install NodeJS from https://nodejs.org/en/download
2. Open another terminal and cd into stock-data-assessmant/frontend/stock-viewer folder
3. Run "npm install" to install the required packages
4. Run "npm run dev" to start the server
5. The frontend is now live on localhost:5173
